# 资源简介

这里是一个 Java 技术栈的学习和积累过程，希望对大家有帮助。

## 缘起

几年没有码代码，感觉手艺有点生疏了，况且现在框架升级这么快。因为近期工作需要，又投入到码代码的过程中，期望通过一些小切入点积累下技术知识点，而不是而百度一下。

目标，在积累技术并运用到项目中的同时，构建一套 Spring Cloud 的项目脚手架工程。

## 工程说明

现在包括以下几个方面，未来将不断丰富和完善。

* MySQL 相关
* Cloud 相关
* NoSQL 相关
